## 개요

- **대상**: 인제군 홈페이지 `합강소식지` 목록 페이지
- **URL 예시**: `https://www.inje.go.kr/portal/inje-news/inje-pr/newsletter`
- **목표**: 목록에서 “합강소식” 각 호(아이템)별로 **PDF 다운로드 링크 정보**를 추출

---

## 목록(LIST) 페이지에서 추출 가능한 항목

### 1) 목록 메타 (페이지 단위)

- **listUrl**: 크롤링한 목록 페이지 URL
- **totalCount**: 총 건수(정수)
  - 선택자: `.boSrchArea .boMeta`
  - 예시(문자열): `총 55건` → 숫자만 파싱해 `55`
- **pageIndex**: 현재 페이지 번호(정수, querystring `pageIndex`)
  - 예시: `?pageIndex=1`
- **pager** (선택): 총 페이지/현재 페이지
  - 선택자: `.pager .pager-num`
  - 예시(문자열): `1/5` → current=1, total=5

### 2) 아이템(호) 단위

목록은 아래 구조의 반복입니다.

- 반복 컨테이너: `ul.boGallery.ebook > li`

각 `li`에서 추출:

- **title**: 호 제목
  - 선택자: `li .boGallery-sbj`
  - 예시: `2026년 2월 합강소식`
- **thumbnailUrl**: 썸네일 이미지 URL(상대경로일 수 있음)
  - 선택자: `li .boGallery-img img`
  - 예시: `/servlet/Thumbnail?i=/upload/board_meta/...jpg&w=398&h=588`
- **thumbnailAlt**: 썸네일 대체 텍스트(이미지 alt)
  - 선택자: `li .boGallery-img img[alt]`
  - 예시: `2026년 2월 합강소식`

#### PDF 다운로드

- **pdfDownloadUrl**: PDF 다운로드 링크(상대경로일 수 있음)
  - 선택자: `li .boGallery-btnGroup a.boGallery-link[href*=\"/egf/bp/board/meta/download\"]`
  - 예시: `/egf/bp/board/meta/download?articleSeq=246247&metaDataID=FIELD_FILE_PDF`
- **pdfArticleSeq**: PDF 다운로드 URL 쿼리 `articleSeq`
  - 예시: `246247`
- **pdfMetaDataID**: PDF 다운로드 URL 쿼리 `metaDataID` (고정값으로 보임)
  - 예시: `FIELD_FILE_PDF`

---

## 비고

- 목록 샘플 기준으로는 **상세(DETAIL) 페이지 진입 링크가 직접 노출되지 않음**(PDF는 meta download URL).
- URL은 상대경로가 많으므로, 저장 시 `https://www.inje.go.kr` 기준으로 절대 URL로 정규화하는 것을 권장합니다.
