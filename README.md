## 인제군청 소식

- **대시보드**: 날씨, 공지사항, 이번 주 행사, 합강소식지 요약·슬라이더
- **푸시 알림**: 주간일정·공지·구인구직·생활장터·자유게시판 토픽별 구독/해제
- **목록·상세**: 공지·구인구직·생활장터·자유게시판·주간일정 목록 및 글 상세·원본 링크
- **합강소식지**: PDF 뷰어로 소식지 열기
- **설정**: 알림 온/오프, 토픽별 알림 on/off
