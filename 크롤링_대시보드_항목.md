# 대시보드 슬라이더 크롤링 추출 항목

## 개요

- **대상**: 인제군 포털 메인 페이지 배너 슬라이더
- **URL**: `https://www.inje.go.kr/portal`
- **기준 HTML**: [크롤링_대시보드.md](크롤링_대시보드.md) (slick-slider 영역)

---

## 슬라이드 한 건(아이템) 추출 항목

- **이미지**: imageUrl, imageAlt  
- **텍스트**: title (div.hidden_txt, 없으면 imageAlt 활용)  
- **링크**: linkUrl (아이템이 `<a>`일 때만), articleSeq (linkUrl에서 파생)  
- **메타**(선택): dataSeq, order  

---

## 선택자 참고

### 1. 반복 컨테이너 및 아이템

| 항목 | 선택자/위치 | 비고 |
|------|-------------|------|
| 슬라이더 컨테이너 | `.slick.slick-initialized.slick-slider` → `.slick-list` → `.slick-track` | - |
| 슬라이드 아이템 | `.slick-track` 직계 자식: `a.sldrEvts-viewItem.slick-slide`, `span.slick-slide` | **제외**: `slick-cloned` 클래스가 있는 항목(복제본) |

### 2. 아이템 단위 추출 항목

| 항목명 | 선택자/출처 | 예시 값 |
|--------|-------------|---------|
| imageUrl | 아이템 내 `img[src]` | `/upload/popupzone/PPAA0001/popupzone_PPAA0001_20220512174711.jpg` → 절대경로 정규화 시 `https://www.inje.go.kr/upload/...` |
| imageAlt | 아이템 내 `img[alt]` | 비법정 도로 매입 안내 |
| title | 아이템 내 `div.hidden_txt` 텍스트 | 비법정 도로 매입 안내 (없으면 빈 문자열 또는 imageAlt와 동일 처리) |
| linkUrl | 아이템이 `<a>`일 때 `href` | `https://www.inje.go.kr/portal/adm/bulletin?articleSeq=180514` (아이템이 `<span>`이면 null) |
| articleSeq | linkUrl 쿼리 파라미터 `articleSeq` (linkUrl에 inje.go.kr 포함 시) | `180514` |
| dataSeq | 아이템 요소의 `data-seq` 속성 | `438` |
| order | slick-cloned 제외 후 DOM 순서(0부터) | `0`, `1`, … |

### 3. linkUrl 유형 참고

| 유형 | 예시 |
|------|------|
| 인제 공지/게시 | `https://www.inje.go.kr/portal/adm/bulletin?articleSeq=180514`, `.../adm/notice?articleSeq=241085` |
| 외부 URL | `https://job.gwd.go.kr/gwjob`, `https://www.oxfamtrailwalker.or.kr/` |
| 이미지 상대경로 | `/upload/image_attach/2026/01/pop_202601270938.jpg` |
| 빈/무의미 | `http://` → null로 저장 권장 |

---

## 비고

- **slick-cloned** 항목은 수집 제외(중복 제거).
- linkUrl이 `http://` 또는 빈 값이면 null로 저장 권장.
- URL은 상대경로를 `https://www.inje.go.kr` 기준으로 절대 URL로 정규화하는 것을 권장합니다.
