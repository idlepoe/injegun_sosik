# 주간일정 크롤링 추출 항목

## 개요

인제군 포털 주간일정 게시판 크롤링 시 추출하는 데이터 항목 정의입니다.  
기준: **상세 페이지** HTML 구조([크롤링_주간일정_DETAIL.md](크롤링_주간일정_DETAIL.md)).

---

## 상세 페이지 추출 항목

- **글 식별/메타**: url, articleSeq, boardCode  
- **게시글 기본 정보**: title, author, registeredAt  
- **본문**: content  
- **첨부파일**: attachmentUrl, attachmentName, fileSeq  
- **이전/다음 글**(선택): prevArticleSeq, prevTitle, nextArticleSeq, nextTitle  

---

## 선택자 참고

### 1. 글 식별/메타

| 항목명 | 선택자/위치 | 예시 값 | 비고 |
|--------|-------------|---------|------|
| 상세 페이지 URL | - | `...weekschedule?articleSeq=246767` | 쿼리 파라미터 포함 |
| articleSeq | `input[name="articleSeq"]`, 팝업 폼 등 | `246767` | 상세 진입 키 |
| boardCode | `input[name="boardCode"]` | `BD000128` | 필요 시 활용 |

### 2. 게시글 기본 정보 (테이블 행)

| 항목명 | 선택자 | 예시 값 |
|--------|--------|---------|
| title | `.skinTb-sbj` | 주간행사계획(2.16.~2.22.) |
| author | `.skinTb-name` | 자치행정담당관 |
| registeredAt | `.skinTb-date` | 2026-02-13 |

### 3. 본문

| 항목명 | 선택자 | 예시 값 |
|--------|--------|---------|
| content | `.skinTb-conts` | 인제군 주간행사계획입니다. (HTML 제거 후 텍스트) |

### 4. 첨부파일

| 항목명 | 선택자/위치 | 예시 값 |
|--------|-------------|---------|
| attachmentUrl | `div.attachFile a[href*="download"]` 의 href | /egf/bp/board/article/download?fileSeq=122324 |
| attachmentName | 위 `a` 태그 텍스트 | (인제군)주간행사계획(0216~0222).xlsx |
| fileSeq | download URL 쿼리 파라미터 | 122324 |

### 5. 이전/다음 글 (선택)

| 항목명 | 선택자/위치 | 예시 값 |
|--------|-------------|---------|
| prevArticleSeq | 이전글 `a[onclick*="goPage"]` 의 goPage 인자 또는 href의 articleSeq | 246500 |
| prevTitle | 이전글 링크 텍스트 | 주간행사계획(2.9.~2.15.) |
| nextArticleSeq, nextTitle | 다음글 td 텍스트가 "다음글이 없습니다."면 없음 | 없음 |

---

## 비고

- **본문**: HTML 태그 제거 후 순수 텍스트로 저장하는 것을 권장합니다.
- **첨부파일**: xlsx 등 실제 파일 다운로드 시 `attachmentUrl` 또는 `fileSeq`를 사용하면 됩니다.
- **목록 페이지**: 목록 HTML 샘플이 확보되면 별도로 "목록 페이지 추출 항목"을 정의해 확장할 수 있습니다.
